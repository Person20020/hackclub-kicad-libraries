# Hack Club KiCad Libraries

This is a KiCad addon library for use in Hack Club programs. It contains commonly used symbols, footprints, and 3D models that are not included in the default KiCad libraries and allows beginners to more easily install these resources.

## Installation

To install this library, follow these steps:
1. In KiCad, go to `Tools` -> `Plugin and Content Manager` and click the `Manage` button in the top right corner.
2. Click the `+` button to add a new repository and enter the following information:
   - **Name**: Hack Club KiCad Libraries
   - **URL**: `https://raw.githubusercontent.com/Person20020/hackclub-kicad-repository/refs/heads/main/repository.json`

## Contributing

When contributing to this library, please follow these guidelines:

- Libraries should follow the [KiCad Library Conventions](https://klc.kicad.org) to the furthest extent possible. As this is not an official KiCad library, some deviations may be allowed if necessary and justified.
